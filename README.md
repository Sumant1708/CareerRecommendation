# 🎯 AI Career Recommender

An intelligent web application that suggests the **top 3 career paths** based on a user's:

- Skills 🛠️
- Interests ❤️
- Education 🎓
- Age 👤

Built using **Machine Learning + Streamlit**.

## 🧠 Features

- Visual bar chart of recommendations
- Fast and user-friendly

## 🛠️ Tech Stack

- Python

- Joblib

---

## 📊 How It Works

1. User enters:
   - Age
   - Education
   - Skills
   - Interests

2. Data is processed and combined

3. ML pipeline predicts probabilities

4. Top 3 careers are displayed
